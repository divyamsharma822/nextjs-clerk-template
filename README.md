# Preview

 # Setup Instructions
### 1. Install dependencies  

   ```
   npm install
   ```

### 2. Configure the `.env.local` file
    
   ```
  NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY = 
  CLERK_SECRET_KEY = 
  
  NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
  NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up
   ```
### 3. Run Next App
  
   ```
   npm run dev
   ```
